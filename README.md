# JH_Study_History

JH_Study_History는 한국사 능력 검정시험 한능검 1급 목표로 만든 학습/문제 풀이 웹 앱입니다. 이 프로젝트는 역사 문제 풀이, 학습 지도, 지역별 역사 정보 제공 기능을 담고 있습니다.

## 주요 기능

- 📝 문제 풀이 모드: 전근대사와 근현대사를 선택해 객관식 문제를 푸는 기능
- 📚 학습 모드: 삼국시대, 통일 신라·발해, 고려, 조선 시대별 역사 지도를 클릭해 관련 정보 확인
- ✅ 정답/힌트 제공: 선택한 문제에 대한 힌트와 정답 보기
- 📈 진행도 표시: 문제 풀이 진행률과 점수를 실시간으로 업데이트

## 사용 방법

1. `index.html` 파일을 브라우저로 열거나 Live Server로 실행합니다.
2. 상단의 `문제 풀이` 탭에서 `전근대사 30` 또는 `근현대사 30` 버튼을 클릭합니다.
3. `학습` 탭을 선택하면 시대별 지도를 보고 관련 역사 정보를 확인할 수 있습니다.

## 파일 구조

- `index.html` - 메인 페이지 구조
- `style.css` - 페이지 스타일
- `script.js` - 퀴즈/학습 모드 로직
- `data.js` - 문제 데이터 및 학습 정보 데이터

## 수정 내용

- 퀴즈 모드 선택 시 문제 데이터가 정상적으로 로드되지 않던 오류 수정
- 탭 전환 시 올바른 이벤트 핸들링 보완
